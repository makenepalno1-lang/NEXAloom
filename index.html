Deploy on GitHub Pages

Your NEXAloom website live for free — step by step, 10 minutes max.

1

Save this file as index.html

Open Notepad (Windows) or TextEdit (Mac). Paste the full website code. Save the file and name it exactly:

index.html

⚠️ Make sure it's .html not .txt — in Notepad choose "All Files" when saving.

2

Create a free GitHub account

Go to github.com → click Sign Up → enter your email, password, username. Verify your email. Done.

3

Create a new repository

Click the green "New" button or the + icon at top right → "New repository". Name it exactly:

nexaloom

Set it to Public. Check "Add a README file". Click "Create repository".

4

Upload your index.html file

Inside your new repository, click "Add file" → "Upload files". Drag your index.html file into the box. Scroll down and click "Commit changes".

5

Enable GitHub Pages

Go to your repository → click Settings (top menu) → scroll down to Pages (left sidebar) → Under "Source" select Deploy from a branch → choose main branch → click Save.

6

Your site is live! 🎉

Wait 2–3 minutes, then visit your site at:

https://YOUR-USERNAME.github.io/nexaloom

Replace YOUR-USERNAME with your GitHub username. Share this link with everyone!

💡 Want nexaloom.com? Buy a domain on Namecheap (~NPR 1,500/yr) and in GitHub Pages settings add it under "Custom domain". Your free hosting + your own .com domain = professional brand.
